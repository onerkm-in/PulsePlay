# Finance / FP&A

The finance / FP&A seat covers margin steering, working-capital management, plan / forecast / actual reconciliation, close anomaly detection, and scenario modelling. In CPG, the finance team carries cross-functional reconciliation responsibility — supply chain, commercial, manufacturing, and procurement decisions all land in the margin bridge.

## What this sub-vertical covers

- **Margin bridge**: price, volume, mix, commodity, FX, plant variance, trade, channel.
- **Working capital cockpit**: inventory, receivables, payables, deduction leakage, accrual accuracy.
- **Customer / category / brand P&L**: dynamic profitability views with cost-to-serve attribution.
- **Close and forecast anomaly detection**: month-end and quarter-close anomaly surfacing.
- **Scenario modelling**: commodity, FX, channel-mix, plant-disruption, retailer-deduction scenarios.
- **CSRD-aligned financial-sustainability reporting** (in cooperation with the Sustainability overlay).

## Why a CPG team uses this

- "Why did gross margin compress 110 bps vs plan in Q3?" — margin-bridge decomposition.
- "Working capital is up $150M vs forecast. What is driving it (inventory build, receivables stretch, payables shift, deduction leakage)?"
- "Close anomaly: revenue in account X looks $4.2M higher than the trend. Real or correction?"
- "If commodity Y rises 12% and FX moves 3% against us, what is the Q4 margin impact by region and category?"

## Typical data sources

- **ERP**: SAP S/4HANA, Oracle Fusion, Microsoft Dynamics — GL, AR, AP, costing, close.
- **EPM / consolidation**: SAP Group Reporting, OneStream, Anaplan, Workday Adaptive Planning, Oracle Hyperion / EPM.
- **TPM (trade)**: see commercial-retail.
- **Treasury / commodity**: Kyriba, FIS Quantum, GTreasury, internal hedging systems.
- **Internal**: financial data warehouse, certified semantic layer.
